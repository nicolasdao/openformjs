# Dev
## Run locally
To run this project locally:

```
npm run dev
```

## Deploy

1. Lint:
```
npm run lint
```
2. Build:
```
npm run build
```
3. Commit changes
4. Check version
```
npm run v
```
5. Bump version:
```
npm run rls <new version>
```
6. Deploy:
```
npm run push
```
